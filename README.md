Unit Converter



A Python GUI application that converts between multiple units of distance, mass, and temperature using Tkinter and Pint.



Features:



Converts units like meters ↔ miles, °C ↔ °F, kilograms ↔ pounds.



User-friendly GUI with dropdowns and entry fields.



Automatically handles unit conversion logic.



Usage:



Clone the repository:



git clone https://github.com/MarkoZivkovic1984/unit-converter.git



Install dependencies:



pip install -r requirements.txt



Run the application:



python converter.py



Example:



Enter a value in Količina.



Select input unit Iz Mere.



Select output unit U Meru.



Click Izračunaj to see the converted value.



Dependencies



Python 3.x



pint



tkinter (built-in with Python)
